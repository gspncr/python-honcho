# python-honcho
Python endpoint to kill, bounce, and start processes. Based on PSUtil: http://psutil.readthedocs.io

## kill a process:
`/kill/<process>`

## bounce a process:
`/bounce/<process>`

## start a process:
`/start/<process>`

## mac bounce a process
`/mac/bounce/<process>`

## mac start a process
`/mac/start/<process>`
